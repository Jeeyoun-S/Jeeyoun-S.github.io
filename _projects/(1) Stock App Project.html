Stock App Project

개요

  • 주제 주식 정보 조회 및 모의주식투자가 가능한 서비스
  • 기간 2022.09.29 ~ ing (SSAFY 교육 진행으로 1월부터 일시 중단)
  • 구성 5인 팀 (8기 SSAFY 교육생)
  • 코드 Backend Repo Frontend Repo

주요 기능

1. 주식 정보 조회
  • 주요 지수, 환율 정보, 카테고리별 종목 주가, 추천 주식을 조회할 수 있습니다.
  • 상세 페이지에서 주가 변동 그래프, 종목 정보, 관련 뉴스, 관련 종목, 실시간 주가를 조회할 수 있습니다.
  • 상장 예정인 주식을 볼 수 있습니다.
2. 주식 및 외환 거래
  • 실시간 주가로 원하는 주식을 모의로 매수 및 매도할 수 있습니다.
  • 실시간 환율로 외환을 원화로, 원화를 외환으로 환전할 수 있습니다.
3. 주식 검색
  • 자카드 유사도로 검색된 주식을 볼 수 있습니다.
  • 검색창에서 최근 검색 기록과 조회수 TOP 5 종목을 볼 수 있습니다.
4. 회원 관리
  • 회원가입 및 로그인을 할 수 있습니다. 사용자에게는 가입 시, 500만 원의 자금이 주어집니다.
  • 관심 주식, 보유 주식, 통장 잔고 조회가 가능합니다.
  • 개인 랭킹, 수익률, 전날 대비 수익률 변화 등을 볼 수 있습니다.

개발 환경

1. Backend
  • Spring Tools 3 for Eclipse
  • Java 8
  • Spring Boot 2.7.4
  • Swagger2 2.9.2
  • Gradle 7.5
  • MyBatis 2.2.2
2. Frontend
  • Android Studio 4.2.2
  • Java 8
  • Gradle 6.7.1
3. Database
  • MySQL 8.0.21

나의 역할

  • Backend 담당
  • 요구사항 명세서 작성 및 API 설계
  • MyBatis를 이용해 회원 관리, 사용자 랭킹, 찜 목록 관리 기능 Rest API 구현
  • Session에 정보를 저장해 주식 및 외환 매매 구현
  • Open API에서 RestTemplate을 이용해 실시간 주가, 환율, 주요 지수 정보 조회
  • 타니모토 유사도와 JSON을 이용한 추천 알고리즘 구현

개발 후기

  • Spring에 대해 처음으로 학습을 진행하며 개발을 했는데, Backend 개발 기초 지식을 쌓을 수 있는 시간이었습니다.
  • RestTemplate, MyBatis, 추천 알고리즘 구현 등 대부분이 처음 해보는 것이라
  • 폭포수 방법론으로 개발을 진행했는데, 이로 인해 개발 진행이 느려지지 않았나 싶어 아쉬움이 많습니다. 또한, 더미 데이터로 테스트용 API를 만들어 Front에 전달했다면 개발 속도에 더 도움이 됐으리라 생각됩니다.
